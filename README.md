# SCIENTIA 6

Official website for the 6th edition of SCIENTIA, the Annual Science Event of Assam University Science Block Departments.

## About

SCIENTIA is a celebration of knowledge, innovation, and cultural diversity. This website showcases the festival's events, gallery, committee members, and schedule.

## Technology Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- GSAP (GreenSock Animation Platform)
- React Router DOM

## Features

- Modern, responsive design
- Interactive animations and transitions
- Event listings and gallery
- Committee information
- Schedule management
- Smooth page navigation
